# Pi Remote Control

React Native app to control OpenClaw agents via remote gateway.

## Features

- 🔐 **Secure Connection** - Connect with Gateway URL + Token
- 💬 **Agent Chat** - Send messages to your AI agents
- 🔊 **Voice Output** - TTS speaks agent responses
- 📊 **Gateway Status** - Monitor connection health

## Tech Stack

- React Native + Expo
- TypeScript
- OpenClaw WebSocket API
- Expo Speech (TTS)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the app:
   ```bash
   npx expo start
   ```

3. Connect to your OpenClaw gateway

## Configuration

Enter your OpenClaw gateway URL and token to connect.
